About This Project

This website is a project created to get familiar with the Angular Framework. It is not intended for real-world use.

Purpose

The goal of this project is to demonstrate knowledge of Angular's core concepts, including:

  • Component-based architecture
  • Client-side routing
  • Integration with RESTful APIs
  • Reusable services
  • State management using RxJS
